Music Uploader

Uploaded folders (Artist / Album / Songs) are written under Library Path. After a batch finishes, Jellyfin's library scan runs so the new tracks show up.

The web-UI button is loaded via a script tag — paste this into Dashboard → General → Branding → Custom CSS: <script src="/musicup/frontend/js/uploader.js" defer></script> An Add Music button then appears in the bottom-right for admin users.

Absolute path on the server. Must already exist and be writable by Jellyfin.
0 disables the cap.
Comma-separated, lowercase, no leading dot.
